Overall

This section presents the overall architecture of the electrochemical sensor circuit platform, including component selection, schematic design, and PCB layout.

The system is designed for electrochemical sensing and can also be adapted for general low-current measurement applications.

Component Selection

The following table summarizes representative component choices used in the current prototype.

Block Selected Part Reason for Selection
Potentiostat Op Amp TBD Stable electrode bias control and analog compatibility
TIA Op Amp OPA380 (TI) Low-noise transimpedance amplification for current readout
ADC TBD Suitable resolution and interface for sensor signal digitization
Reference / Bias TBD Stable bias generation for electrochemical operation
MCU / Controller TBD Data acquisition, control, and interface
